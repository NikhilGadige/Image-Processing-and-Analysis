## Problem Description:

In this experiment, we were asked to design an algorithm that converts a given input image into different spatial and intensity resolutions, stores the result in a custom binary file format or matrix (.mat) with a compact header, and then reconstructs the image using a decoder.

## Running Instructions

1. Run the encoder.m.
2. Then : Select the image -> Choose spatial and intensity resolutions -> The encoded binary file is generated.
3. Run the decoder.m.
4. The decoded image is reconstructed and displayed.
