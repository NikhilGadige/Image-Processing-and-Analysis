# Image Processing and Analysis - Semester 6 Lab Submissions

[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/NikhilGadige/Image-Processing-and-Analysis)

This repository contains the complete set of lab submissions for the Semester 6 course **Image Processing and Analysis**.

Each lab includes MATLAB programs, related input/output assets (where applicable), and a dedicated README describing objectives, methodology, and observations.

## What This Repository Contains

- Practical implementations of core digital image processing concepts taught during the course.
- Step-by-step experimentation across spatial domain, frequency domain, restoration, and color processing topics.
- Lab-wise code organization to keep experiments modular, reproducible, and easy to evaluate.

## Lab Overview

- `Lab_01/` - Introductory exercises focused on encoding and decoding operations.
- `Lab_02/` - Geometric image manipulation using affine transformation techniques.
- `Lab_03/` - Intensity transformation and enhancement methods including negative, log, gamma, contrast stretching, and histogram processing.
- `Lab_04/` - Frequency domain analysis and processing using 2D DFT and FFT-based implementations.
- `Lab_05/` - Image degradation and restoration experiments, including noise analysis and filtering-based recovery.
- `Lab_06/` - Color image processing tasks and channel-level operations.

## Directory Notes

- Every lab folder has its own `README.md` with detailed experiment-level explanations.
- Supporting folders such as image datasets and output directories are included where required by that lab.

## Academic Context

This repository is maintained as part of the official laboratory submission record for the **Image Processing and Analysis** course in Semester 6.
