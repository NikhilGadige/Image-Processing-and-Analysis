# Problem Statement:

In this assignment, we are asked to design and implement a program to perform an affine transformation on a given input image. The program should allow the user to specify transformation parameters and generate the transformed output image.

## User Inputs includes:

The program must prompt the user to enter the following parameters:
1. Horizontal scaling factor  (range more than 0)
2. Vertical scaling factor (range more than 0)
3. Rotation angle (ranging from -180 to +180 degrees)
4. Horizontal translation (Any integer (in pixels))
5. Vertical translation (Any integer (in pixels))
6. Horizontal shear factor (ranging from -1 to +1)
7. Vertical shear factor (ranging from -1 to +1)

## Running Instructions

1. Run the Affine_Transformation.m.
2. When prompted:
    - Select the image.
    - Enter the required affine transformation parameters.
3. The affine transformation is applied to the image.
4. The original image and the transformed image are displayed.
